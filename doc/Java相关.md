## 排序
* 内部排序：归并排序、交换排序（冒泡、快排）、选择排序、插入排序等。
* 外部排序：掌握利用内存和外部存储处理超大数据集，至少要理解过程和思路。

考察算法不仅仅是如何简单实现，面试官往往会刨根问底。
* 比如那些是不稳定的？（快排、堆排）
* 或者思考稳定意味着什么？
* 对于不同数据集，各种排序的最好或最差情况？
* 从某个角度如何进一步优化（比如空间占用，假设业务场景需要最小辅助空间，这个角度堆排就比归并优异）？
* 从简单了解到进一步思考，能观察面试者处理问题和沟通时的思路。
